Recently a friend was having trouble developing a custom Shopify theme, specifically getting Sass to compile to Liquid when the Sass file contains Liquid variables.

Using Sass interpolation you can easily add Liquid variables.

```Sass

h1 {
  color: #{'"{{ header_color }}"'}
}

```
